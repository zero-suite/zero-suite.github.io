Vault Zero

One vault for every API key you own

Unlock once. Generate .env files for any project in any framework. Your secrets never leave your machine.

Framework-aware naming

The same key gets the right prefix automatically. NEXT_PUBLIC_* becomes EXPO_PUBLIC_* or VITE_* depending on your project. No manual renaming.

Agent identity scoping

Create a time-limited credential set for Claude Code or Cursor. Your Stripe key stays invisible. The .env expires automatically.

Secret scanner

Scans your code and git history for leaked keys. 60+ patterns. Exits with code 1 — CI fails automatically if you commit a secret.

Team sync

Encrypted vault shared via any git remote. Your git host never sees plaintext secrets. Works with GitHub, GitLab, or self-hosted.
