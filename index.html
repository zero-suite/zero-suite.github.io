Features

Everything a solo founder needs

Built for the developer who runs 3-10 products with 50+ API keys across dev, staging, and production.

🔐

XChaCha20 + Argon2id encryption

Military-grade encryption. Your secrets never leave your machine unencrypted. No cloud dependency, no vendor lock-in.

⚡

Framework-aware .env generation

One vault, any framework. NEXT_PUBLIC_* becomes EXPO_PUBLIC_* or VITE_* automatically. Works with Next.js, Expo, Vite, SvelteKit, Nuxt, Remix, Astro, and Node.

🤖

AI Agent Identity Layer

Give Claude Code only the keys it needs. Create scoped, time-limited credential sets per agent — Stripe stays invisible to your coding assistant.

⚡

Stack Bootstrap

Tell vz your stack. It tells you exactly which secrets you have, which you're missing, and where to get them — with links.

🔍

Secret Scanner

Scan your entire codebase and git history for leaked credentials. 60+ patterns. Exits with code 1 so CI pipelines fail automatically.

👥

Team Sync

Share encrypted vaults with your co-founders via any git remote. XChaCha20 encrypted — your git host never sees plaintext secrets.

💊

Vault Health Score

A to F grade for your credential hygiene. Tracks rotation schedules, overdue keys, miscategorized publishable keys, and more.

🔄

Rotation Tracking

Set rotation intervals per secret. Get reminders when keys are overdue. Never get surprised by an expired API key in production again.

🔌

49 built-in providers

Cost tracking, deploy triggers, log tailing, and uptime monitoring across every major platform — all from one CLI.
