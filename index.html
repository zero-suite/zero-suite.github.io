01 — Vault Zero

Stop copying .env files
between projects

One encrypted vault for every API key you own. Unlock once, generate .env for any project in any framework.

vault demo
$ vz import .env.production
✓ 12 secrets imported
  Anthropic  Supabase  Clerk  Stripe  Convex  Resend

$ vz env --framework expo
✓ .env written
  NEXT_PUBLIC_SUPABASE_URL  →  EXPO_PUBLIC_SUPABASE_URL
  NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY  →  EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY

$ vz health
Score: 95/100  Grade: A   12 secrets  0 overdue rotations

$ vz scan ./src --critical-only
⚠ CRITICAL  Stripe key in src/old-config.js:14  →  rotate now

Framework-aware naming

The same secret automatically gets the right prefix for Next.js, Expo, Vite, SvelteKit, Nuxt, Remix, Astro, or Node. No manual renaming.

Agent identity scoping

Create a scoped, time-limited .env for Claude Code or Cursor. Your Stripe key stays invisible. Expires automatically.

Secret scanner

60+ patterns. Scans files and git history. Exits with code 1 so CI fails if you accidentally commit a key.

Team sync

Encrypted vault shared via any git remote. Your git host never sees plaintext. Works with GitHub, GitLab, or self-hosted.

02 — Deploy Zero

Deploy from the terminal,
or let Claude do it

Ten platforms. One CLI. The MCP server lets your AI coding agent trigger deploys directly — no browser required.

deploy demo
